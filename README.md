# 〈참고1〉 AI챔피언 인증 체계

○ **(교육·인증연계)** AI·데이터 중심으로 대상·수준별 4단계 공공 AI역량 트랙과 AI챔피언을 연계하여 운영

## 〈 행정안전부 공공 AI역량 트랙(4단계) 커리큘럼 〉

| (트랙1) AI활용 | **(트랙2) AI융합** | **(트랙3) AI개발** | (트랙4) AI마스터 |
|:---:|:---:|:---:|:---:|
| 전직원 대상 | 실무자 대상 | IT담당 대상 | IT담당(고급) 대상 |
| | **AI챔피언 그린** | **AI챔피언 블루** | |
| ① AI 리터러시와 업무 활용 | ④ 생성형 AI 활용 노코드 데이터분석 | ⑤ AI 행정 융합 기획 *(트랙 2, 3 공동 활용)* | ⑨ AI챔피언 고급 *(분석기술교육 외)* |
| ② 데이터 리터러시 | ⑤ AI 행정 융합 기획 | ⑦ 생성형 AI 활용 데이터분석 심화 | |
| ③ 관리자 AI 리더십 | ⑥ 노코드 AI 서비스 구현 | ⑧ 바이브코딩 LLM 서비스 개발 | |

---

○ **(과정 목표)** 행정·공공기관 직원을 대상으로, AI기술을 활용한 **실무 과제 수행**(수행평가)을 통해 문제해결 역량을 종합적으로 강화·인증

> AI융합과정 요구역량 달성자는 **그린** 인증 / AI개발과정 달성자는 **블루** 인증

| 교육과정 | 육성 목표 | AI챔피언 연계 | 역 할 |
|:---:|:---:|:---:|:---|
| **[트랙2] AI 융합** | **AI융합 실무자** | **그린** | AI 전환 아이디어를 정책·서비스 기획안으로 구체화할 수 있는 실무 기획자 |
| **[트랙3] AI 개발** | **AI전환 실행자** | **블루** | AI 전환 사업을 기획·설계하고, PoC부터 구축·준공까지 실무 전과정을 주도하는 실행 중심자 |

---

○ **(인증 방식)** 3가지 방식(①교육과정형, ②자기주도형, ③자격연계형)으로 인증 부여

> ① **교육과정형: 교육 + 수행평가를 통해 인증**  
> ② **자기주도형:** 수행평가만 참여하여 인증  
> ③ **자격연계형:** 지정 자격증 취득+시정 과목 이수 시 인증
