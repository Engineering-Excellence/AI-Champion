# 과목 3. 자동화 산출물 – 제출물

## 사용 AI 도구
- **Claude Code (Opus 4.8)**: 데이터 구조 파악, `solution.py` 작성, 정적 검색 페이지(HTML/JS) 구현.

## 구현 메모
1. **산출물 ① 정적 페이지** (`산출물1_정적페이지/index.html`): 500건 사업 데이터를 `data.js`로 임베드해 사업명·지역·사업분야·총사업비를 카드로 표시하고, 사업분야/지역/사업명으로 필터링 및 `결과 N건`을 실시간 표시. 첨부 카드뉴스 3장과 korea.kr 정책뉴스 RSS를 임베드(라이브 프록시 로드 실패 시 빌드 시점 스냅샷 `rss.js`로 최신 8건 폴백)했다.
2. **산출물 ② 자동화 도구** (`solution.py`): CSV 2종(UTF-8 BOM)을 읽어 총 사업 수·완료 수·평균 총사업비·사업분야별 건수·지역별 재정자립도 상위 3을 콘솔과 `결과_요약.csv`로 출력. 실행 결과 총 500건 / 완료 295건 / 평균 148억 원.
3. **배포**: Netlify에 정적 페이지 배포. `solution.py`는 로컬에서 실행 가능.
