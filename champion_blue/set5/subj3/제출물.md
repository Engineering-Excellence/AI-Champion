# 제출물 — [연습세트05·블루] 3과목 자동화

## 사용 AI 도구
- **Claude (Claude Code)** — 정적 페이지(HTML/CSS/JS)와 `solution.py` 작성, 실행 검증.

## 구현 메모
1. **정적 페이지(`site/index.html`)** — `장데이터.json`을 `fetch`로 읽어 장별 카드를 렌더링하고, 키워드 입력 시 제목·본문을 필터링하며 "결과 N건"과 일치 부분 하이라이트를 표시한다.
2. **외부 리소스** — 보고서 5장 사실 정합성 매트릭스를 Google Sheets `<iframe>`으로 임베드(배포 시 `SHEETS_EMBED_URL`만 교체). 시트 미연결 시에도 동일 내용을 보여주는 정적 HTML 매트릭스를 함께 제공한다.
3. **Python 도구(`solution.py`)** — 표준 라이브러리(`csv·json·sys·pathlib`)만 사용. 키워드를 인자로 받아 본문 포함 장만 추려 `결과_<키워드>.csv`(헤더 장·제목·본문)로 저장하고 `[OK] N건 저장 → ...`을 출력한다.
4. **검증** — `python solution.py "점검"` 실행 → 5개 장 모두 "점검" 포함, `결과_점검.csv`(5건) 생성 확인.

## 배포
- 배포는 직접 진행 예정이라 `배포_URL.txt`는 비워 둠. (`site/` 폴더를 Netlify/Vercel에 업로드 후 URL 기입)
