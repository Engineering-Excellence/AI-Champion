# [연습세트01·블루] 3과목 — 자동화 제출물

## 사용 AI 도구
- **Claude (Claude Code)** — 데이터 분석, `solution.py` 작성, 정적 페이지(`index.html`) 생성에 활용.

## 구현 메모
1. **산출물 ① 정적 페이지(`index.html`)**: 첨부 `청렴신고_사건데이터.csv` 500건을 JSON으로 임베드한 단일 HTML. 신고유형·처리상태 드롭다운 필터와 키워드(지역·사건ID) 검색, "결과 N건" 표시, 사건 카드(신고일자·신고유형·지역·처리상태)를 제공. Netlify/Vercel에 그대로 업로드하면 배포된다.
2. **외부 리소스(필수)**: 행안부 보도자료 RSS(`https://www.korea.kr/rss/dept_mois.xml`)를 CORS 프록시 경유로 가져와 최신 보도자료 5건 이상을 페이지 하단에 임베드.
3. **산출물 ② Python 도구(`solution.py`)**: 표준 라이브러리(`csv`·`pathlib`)만 사용. 두 CSV를 읽어 총 신고 수, 처리완료 수, 평균 처리기간, 신고유형별 건수, 기관유형별 평균 이수율을 콘솔과 `결과_요약.csv`에 출력. 빈 값(`처리기간_일`·`교육이수율`)은 평균 계산에서 제외.
4. **검증 결과**: `python solution.py` 실행 시 `총 신고 500건 / 처리 완료 229건 / 평균 처리 52일`로 문제지 예시 출력과 일치.

## 제출 파일
- `index.html` — 산출물 ① 정적 검색·필터 페이지 (배포_URL.txt 대체)
- `solution.py` — 산출물 ② Python 자동화 스크립트
- `결과_요약.csv` — solution.py 실행 결과
- `제출물.md` — 본 문서
