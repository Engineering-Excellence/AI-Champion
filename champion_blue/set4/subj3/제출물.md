# 제출물 — [연습세트04·블루] 3과목 자동화

## 사용 AI 도구
- Claude (Claude Code, Opus 4.8)

## 구현 메모
- **산출물 ① 정적 페이지(`site/index.html`)**: 안내서비스 24건을 카테고리(7개)로 묶어 카드로 표시했다. 각 카드는 카테고리명·건수(배지)·대표 문의기관(최빈값)·서비스 목록을 보여준다. 카드뉴스 이미지 4장을 페이지 상단에 `<img>`로 임베드했고(필수 3장 충족), 데이터는 HTML에 인라인으로 담아 별도 서버 없이 동작한다.
- **키워드 검색**: 입력값을 카테고리명 또는 문의기관과 대소문자 무시로 매칭해 실시간 필터링하며 "결과 N건"을 표시한다.
- **산출물 ② Python 도구(`solution.py`)**: 표준 라이브러리(`csv`·`pathlib`·`collections`)만 사용한다. `안내서비스.csv`(UTF-8 BOM)를 읽어 카테고리별로 `출력_카테고리별/<카테고리명>.csv`로 분리 저장하고, 카테고리별 건수·대표기관 집계를 `결과_카테고리집계.csv`로 저장하며 콘솔에도 출력한다.
- **실행 결과**: 총 24건 → 민원5·복지4·교육3·교통3·세금3·일자리3·재난3.
