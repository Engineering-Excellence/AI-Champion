# 제출물 메모

- **사용 AI 도구:** Claude (Claude Code)
- **산출물 ① 정적 페이지(`index.html`):** `신청목록.json` 데이터를 인라인으로 포함해 fetch/CORS 없이 어떤 호스팅에서도 동작하도록 구현. 지자체·사업명·예산을 카드로 표시하고, **한도액 = 인구수 × 10천원/명** 규칙으로 예산 초과 여부를 승인(녹색)/반려(빨강)로 자동 판정. Google Sheets 임베드용 `<iframe>` 자리 포함(웹에 게시 URL만 넣으면 됨).
- **산출물 ② 자동화(`solution.py`):** `첨부/신청서/*.md`의 메타 헤더 `<!-- 지자체: ___ -->`를 정규식으로 파싱해 `출력_지자체별/<지자체명>/`로 복사하고, 콘솔 출력 + `결과_분류현황.csv`(utf-8-sig) 저장. 실행 결과: 신청서 10건 → 지자체 10곳 각 1건.
- **자동 판정 기준 가정:** 문제지에 한도액 명시가 없어 1인당 10천원 기준을 채택(상수 `PER_CAPITA`로 한 줄 변경 가능). 결과 승인 7건 / 반려 3건(나봄시·라윤시·사계구).
- **남은 수동 작업:** Netlify Drop 또는 Vercel로 `index.html` 배포 후 `배포_URL.txt`에 URL 기입, Sheets 임베드 URL을 iframe `src`에 삽입.
