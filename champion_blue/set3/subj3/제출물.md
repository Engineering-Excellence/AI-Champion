# 제출물 — 3과목 자동화 (연습세트03·블루)

## 사용 AI 도구
- **Claude (Claude Code / Opus 4.8)** — 정적 페이지(`site/index.html`)와 Python 도구(`solution.py`) 작성·검증

## 구현 메모
- **산출물 ① 정적 페이지** (`site/index.html`): `조항데이터.json` 15개 조항을 카드로 표시하고, 제목·본문 기준 키워드 검색 + "결과 N건" 카운트 + 매칭어 하이라이트를 제공. 카드뉴스 이미지 4장(`이미지/카드뉴스1~4.png`)을 `<img>` 태그로 상단에 임베드(필수 3장 이상 충족). 데이터는 `<script type="application/json">`으로 인라인 임베드해 `file://` 로컬 실행과 배포 모두에서 동작.
- **산출물 ② Python 도구** (`solution.py`): 표준 라이브러리(`csv·json·sys·pathlib`)만 사용. 명령행 키워드를 받아 `첨부/조항데이터.json`에서 본문 또는 제목에 키워드가 포함된 조항만 추려 `결과_<키워드>.csv`(UTF-8 BOM, 헤더 조·제목·본문)로 저장하고 건수를 출력.
- **실행 예시**: `python solution.py "기본계획"` → `[OK] 2건 저장 → 결과_기본계획.csv`
